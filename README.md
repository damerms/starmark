# Starmark Investment Management — Platform Showcase
*Mobile-first UX/UI Design & Information Architecture*

---

### Project Overview
This repository contains the front-end codebase and design architecture for the **Starmark Investment Management** digital presence. Starmark is an FCA-regulated investment group providing regulatory umbrella services and fund management structures for professional counterparties.

### Portfolio Status
This project is currently **Archived** and is intended for **showcase purposes only**.
* **Status:** Completed
* **Primary Focus:** Responsive UX/UI, Financial Services Branding, and Regulatory Information Architecture.
* **Maintenance:** This repository is no longer actively maintained or open for external contributions.

### Key Contributions
* **Design Leadership:** Architected the visual identity and interface for the Starmark Group, ensuring compliance with financial sector standards.
* **Information Architecture:** Streamlined complex regulatory content into a mobile-first, digestible format.
* **Technical Logic:** Implemented modular components to support scalability across Starmark Holdings Limited and its subsidiaries.
